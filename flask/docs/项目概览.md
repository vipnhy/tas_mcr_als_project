# TAS MCR-ALS 分析平台 - 项目概览

## 🎯 项目简介
这是一个专业的瞬态吸收光谱(TAS)数据分析Web平台，基于多元曲线分辨-交替最小二乘法(MCR-ALS)算法，为研究人员提供简单易用的在线分析工具。

## ❓ 常见问题

**Q: 支持什么数据格式？**
A: CSV、TXT、DAT格式，最大50MB

**Q: 如何选择组分数量？**
A: 建议从2-3个开始，根据LOF值调整

**Q: 分析需要多长时间？**
A: 通常1-5分钟，取决于数据大小

**Q: 如何判断结果好坏？**
A: LOF < 10%为良好，结合光谱合理性判断
